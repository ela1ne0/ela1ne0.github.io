HackDuke 2026 Visual Identity

HackDuke is a 2,000+ attendee annual hackathon at Duke University. For the 2026 event, our 7-person design team created a cohesive brand identity that extended across web, merchandise, and physical installations—while improving registration performance.

RoleBrand Designer
Team7 designers
Timeline6 months
ToolsFigma, React, Illustrator
View website →

Overview

I focused on translating our visual direction into a scalable system and collaborating with the development team so the final website felt expressive but still clear, usable, and mobile-first.

Context & Goals

When we began, previous HackDuke branding felt fragmented. The visual language didn’t carry consistently across platforms, and the website hierarchy made calls-to-action less prominent than they should have been.

  • Create a memorable visual identity.
  • Improve website hierarchy and CTA clarity.
  • Build a system future teams could reuse.
  • Increase registrations year-over-year.

Concept Development

We explored multiple directions—minimal and futuristic, playful and illustrative, and a narrative-driven system built around a central character. We chose a character-driven approach because it created a strong anchor across touchpoints and made the event instantly recognizable.

I worked on refining the character system and defining how it would interact with typography and layout elements.

Building the System

Rather than designing a single set of static pages, we built a modular system that could scale across web, print, and environmental design.

Website Execution

I collaborated closely with developers to translate designs into React components. We iterated on hierarchy and spacing to ensure the most important actions were obvious, especially on mobile.

  • Strengthened CTA contrast and placement.
  • Simplified the registration path.
  • Improved mobile spacing and readability.
  • Reduced visual clutter while keeping personality.

Impact

  • ~1,000 registrations (vs ~500 previous year)
  • ~60% increase year-over-year
  • Consistent visual ecosystem across 10+ touchpoints
  • Reusable system delivered for future HackDuke teams

Reflection

This project pushed me to design beyond individual screens and toward system-level thinking. I learned how to balance expressive branding with usability, and how to collaborate across design and development without losing cohesion.
